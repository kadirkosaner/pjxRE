# Emma Level 1 - Image Prompts (Nano Banana Format)

Aşağıdaki liste, Emma'nın Level 1'deki 3 farklı aşaması (Common, WorkingOnBreak, WorkingDone) altındaki toplam 27 konu (topic) için özel olarak hazırlanmış Nano Banana (Flux) promptlarını içermektedir.

**Hatırlatma:**

- Karakterin kameraya **bakmaması**, çevreyle veya olaylarla etkileşimde olması amaçlanmıştır.
- Senin birleştireceğin arka planlara uygun olması için mekanlar (Diner İçi, Mutfak, Bulaşıkhane vs.) sadece atmosfer/açı bilgisi olarak verilmiş, sahne detaylarından çok **Emma'nın yüz ifadesi, duruşu ve ışığına** odaklanılmıştır.
